## ⏳ НАСТУПНИЙ КРОК:

### 🌐 Крок 3: Деплой на Render.com

**Інструкція:** [`КРОК_3_RENDER.md`](КРОК_3_RENDER.md)

#### Що потрібно зробити:

1. **Увійти на Render** (1 хв)
   - https://dashboard.render.com
   - Sign in з GitHub

2. **Створити Web Service** (2 хв)
   - New + → Web Service
   - Обрати sys-plus-bot

3. **Налаштувати** (2 хв)
   - Name: sys-plus-bot
   - Region: Frankfurt
   - Build: pip install -r requirements.txt
   - Start: python app.py

4. **Додати змінні** (3 хв)
   - TELEGRAM_BOT_TOKEN
   - KEYCRM_API_KEY
   - KEYCRM_BASE_URL
   - TELEGRAM_ADMIN_IDS (всі 3 ID)
   - PORT=5000

5. **Запуск** (1 хв)
   - Create Web Service
   - Зачекати 2-5 хвилин
